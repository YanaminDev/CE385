# workshop1
ในการดึงข้อมูลทั้งหมดเราใช้ get เเละเส้น endpoint /students เพื่อการดึงข้อมูล เเละสั่งให้ response ส่ง arrey ของ student ไปทั้งหมด

การเอาข้อมูลจาก studentห มาบางตัวเราจะใช้ get เเละส่งพารามิเตอร์ :id มาเพื่อใช้ในการกรองจาก student เราจะเเปลง id ที่ส่งมากับ url มาเเปลงเป็นตัวเลข เเละทำการ find โดยใช้ id ที่รับมา กับ ตัวเเปร array ของ students เเละมาเก็บในตัวเเปร student เเล้วให้ response กลับไป
ถ้าไม่เจอก็ให้ error 404 ด้วยการ return

ในการเพิ่มข้อมูลใหม่ เราจะใช้ post  โดยใช้ endpoint  /students  เเละรับ  object ตัวเเปร name , age  จากนั้นเราจะหาจำนวนของ array ทั้งหมด มาบวกกับ 1 เพื่อใช้สร้างข้อมูลใหม่ จากนั้นสรา้งตัวเเปร  newStudent เพราะสร้าง object ของข้อมูลใหม่โดยมีข้อมูล id , name , age  เเล้วเรา push เข้า students เเละให้ response กลับไป ของข้อมูลทั้งหมด

ในการอัปเดตข้อมูล เราจะใช้ put  โดยใช้ endpoint  /students/:id  เพื่อรับ id ที่ต้องการ เเล้วเอา id นั้นมาลบ หนึ่งเพื่อเอาข้อมูลในตำเเหน่ง ที่เราต้องการจริง เเละรับข้อมูล name age จากนั้นทำการ หาข้อมูลของนักเรียนที่เราต้องการเปลี่ยนด้วยการ find เปรียบเทียบ id กับที่เรารับมาจากพารามิเตอร์เเละในข้อมูล  เเละเช็คเงื่อนไขว่าถ้าเจอข้อมูล ก็ให้อัปเดตข้อมูล ตาม id โดยเอาข้อมมูลจาก name age ที่รับมา มาเเทนที่ข้อมูลเก่า


ในการลบข้อมูลเราจะรับ id มาโดยใช้ endpoint  /students/:id  เเล้วเอา id นั้นมาลบ หนึ่งเพื่อเอาข้อมูลในตำเเหน่ง ที่เราต้องการจริง  เเล้วก็นำมาลูป find เปรียบเทียบ id ที่รับมา  ถ้ามีก็ให้มีการ pop ข้อมูลออกมาจาก students เเล้วทำการ response ข้อมูลทั้งหมดกลับไป
